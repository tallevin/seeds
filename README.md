# Seeds

AI-powered markdown writing application - "Figma for writing"

## Features

- **Rich Markdown Editor**: Full-featured block-based editor with TipTap
- **Document Tabs**: Multi-document editing with tab management
- **Context Panel**: Manage context files for AI suggestions
- **Library**: Browse and organize your reference documents
- **AI Seeds**: Smart suggestions drawn from connected databases
- **Frameworks**: Apply thinking frameworks (McLuhan's Tetrad, Six Thinking Hats, etc.)
- **Role Definition**: Configure AI personality and writing style
- **Ghost Autocomplete**: AI-powered writing suggestions inline
- **Highlight System**: Visual indicators for seed-relevant text blocks

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19 + TypeScript
- Vite
- TailwindCSS v4
- TipTap Editor
- Lucide Icons

## Build

```bash
npm run build
```
